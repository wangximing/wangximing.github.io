Ionic

Ionic简介

Ionic是一个开源的前端开发框架，可以用HTML,CSS,JS来开发Hybird App.

Ionic Stack

  • AngularJS
  • NodeJS
  • Sass
  • Bower
  • Cordova
  • Gulp
  • Travis CI
  • UI-Router

Start building with Ionic

  • Install

    npm install -g cordova ionic

  • Create

    ionic start myApp tabs

  • Run

    ionic serve
